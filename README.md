💜 Empathica — Emotional Wellness Chatbot

A beautifully designed, dark-themed Streamlit web app powered by Claude AI that provides empathetic emotional support.

🌟 Features
🫀 Emotion Detection
Automatically detects the user’s primary emotion (e.g., stress, anxiety, depression) from input text
💬 Topic Identification
Identifies the core theme behind the user's message
👩‍⚕️ Therapist Recommendation
Suggests a relevant therapist profile based on emotion and topic
💜 Empathetic Responses
Generates warm, psychologically grounded replies
🧠 Context-Aware Chat
Maintains full chat history within a session
✨ Starter Prompts
Provides predefined prompts to help users begin
🌙 Dark-Themed UI
Clean and visually soothing interface
🛠️ Tech Stack
Frontend & UI: Streamlit
AI Model: Claude AI
Backend: Python
Environment: Google Colab / Local
🚀 How It Works
User enters a message
System detects emotion and topic
AI generates response + therapist suggestion
Chat history is maintained
